pre-commit.sh

Comprehensive pre-commit validation with PHP syntax, Pint formatting, PHPStan analysis, and security scanning.

Overview

The pre-commit hook runs before git commit commands to ensure code quality. It validates PHP syntax, applies formatting, runs static analysis, and checks for security issues.

Checks Performed

PHP Checks

  1. Syntax Check - Validates PHP syntax with php -l
  2. Laravel Pint - Auto-formats code and re-stages files
  3. PHPStan - Static analysis for type errors
  4. Security Scan - Detects debug functions and hardcoded secrets

Blade Checks

  • Missing @csrf in forms
  • Unescaped output ({!! !!}) warnings

Migration Checks

  • Destructive operations (dropColumn, dropTable, truncate)
  • Missing down() method

Environment Checks

  • Blocks .env file commits

Installation

Add to .claude/settings.json:

{
  "hooks": {
    "UserPromptSubmit": [
      {
        "matcher": "git commit",
        "command": "bash .claude-plugins/laravel-agent/hooks/scripts/pre-commit.sh"
      }
    ]
  }
}

Configuration

Environment variables:

# Number of parallel jobs
export PARALLEL_JOBS=4

# Skip specific checks
export SKIP_PHPSTAN=1
export SKIP_PINT=1

Requirements

Optional but recommended:

# Code style
composer require laravel/pint --dev

# Static analysis
composer require phpstan/phpstan --dev

Exit Codes

CodeMeaning
0Success - all checks passed
1Warning - non-blocking issues
2Error - blocks the commit
