Quick Start

1

Build a Feature

The intelligent build command analyzes your request and decides the best approach:

/laravel-agent:build user authentication with email verification
2

Create an API

Generate versioned REST APIs with documentation:

/laravel-agent:api:make products --versioned
3

Generate Tests

Create comprehensive Pest tests for any class:

/laravel-agent:test:make App/Services/PaymentService
4

Security Audit

Run OWASP security audit on your application:

/laravel-agent:security:audit

Architecture

How It Works

User Request | v +-------------+ | Architect | --- Analyzes request, decides implementation +-------------+ | v +-------------------------------------------+ | Builder Selection | +-------------+-------------+---------------+ | Feature | Module | Service | | Builder | Builder | Builder | +-------------+-------------+---------------+ | v +-------------+ | Testing | --- Generates Pest tests +-------------+ | v +-------------+ | Review | --- Code quality check +-------------+

Decision Matrix

Request Type Implementation Location
CRUD + UI + API Feature app/Features/<Name>/
Reusable logic, no UI Module app/Modules/<Name>/
Orchestrates operations Service app/Services/
Single operation Action app/Actions/<Domain>/
