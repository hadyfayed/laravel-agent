# Install the plugin
/plugin install laravel-agent@hadyfayed-laravel-agent

Installation

Prerequisites

  • Claude Code CLI installed
  • Laravel project (8.x or higher)

Install via Plugin Marketplace

/plugin marketplace add hadyfayed/laravel-agent
/plugin install laravel-agent@hadyfayed-laravel-agent

Verify Installation

After installation, verify by running:

/laravel-agent:patterns

Quick Start

Build a Feature

The intelligent build command analyzes your request and decides the best approach:

/laravel-agent:build user authentication with email verification

Create an API

/laravel-agent:api:make products --versioned

Generate Tests

/laravel-agent:test:make App/Services/PaymentService

Security Audit

/laravel-agent:security:audit

Commands

42 commands organized by category for all your Laravel development needs.

Build Commands

/build Intelligent build - architect analyzes and decides implementation approach
/feature:make Create a complete Laravel feature directly
/module:make Create a reusable domain module with contracts, services, DTOs
/service:make Create a service class or action with proper patterns

API Commands

/api:make Create a versioned API resource with documentation
/api:docs Generate API documentation

Testing Commands

/test:make Generate comprehensive tests for a class or feature
/test:coverage Run tests with coverage report

DevOps Commands

/deploy:setup Configure deployment for your Laravel application
/cicd:setup Configure CI/CD pipeline
/security:audit Run security audit on your application
