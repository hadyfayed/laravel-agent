Agents

23 specialized agents that handle different aspects of Laravel development

laravel-architect

Core

Senior architect that analyzes requests, decides implementation approach (Feature/Module/Service/Action), enforces SOLID/DRY, and delegates to specialized builders.

Delegates to: All builder agents

laravel-feature-builder

Builder

Builds complete features with CRUD operations, views, API endpoints, migrations, and tests in a self-contained structure.

Creates: app/Features/<Name>/

laravel-module-builder

Builder

Builds reusable domain modules without routes/views. Pure business logic that can be shared across features.

Creates: app/Modules/<Name>/

laravel-service-builder

Builder

Creates services for orchestrating operations and actions for single discrete operations with proper patterns.

Creates: app/Services/ or app/Actions/

laravel-api-builder

Builder

Builds REST APIs with versioning, OpenAPI documentation, rate limiting, and proper resource transformations.

Creates: routes/api.php, app/Http/Controllers/Api/

laravel-package

Builder

Creates distributable Laravel packages with proper structure, service providers, and Packagist publishing.

Creates: packages/<vendor>/<name>/

laravel-livewire

Frontend

Creates Livewire 3 reactive components with forms, tables, modals, polling, and real-time updates.

Creates: app/Livewire/, resources/views/livewire/

laravel-filament

Frontend

Builds Filament admin panels with resources, pages, widgets, relation managers, and custom actions.

Creates: app/Filament/Resources/

laravel-auth

Backend

Sets up authentication and authorization with policies, gates, roles/permissions using Sanctum or Passport.

Creates: app/Policies/, config/auth.php

laravel-database

Backend

Creates migrations, optimizes queries, fixes N+1 problems, designs schemas, and manages relationships.

Creates: database/migrations/

laravel-queue

Backend

Creates queued jobs, events, listeners, and notifications with proper retry logic and failure handling.

Creates: app/Jobs/, app/Events/, app/Listeners/

laravel-reverb

Backend

Sets up Laravel Reverb WebSocket server for real-time features with broadcasting configuration.

Creates: config/broadcasting.php

laravel-pennant

Backend

Creates feature flags and A/B tests using Laravel Pennant with proper scoping and resolution.

Creates: app/Features/

laravel-ai

Backend

Creates AI-powered features using Prism PHP for LLM integration with proper prompting and response handling.

Creates: app/Services/AI/

laravel-migration

Backend

Handles Laravel/PHP version upgrades, legacy migrations, and codebase modernization with Rector.

Uses: rector, php-cs-fixer

laravel-performance

Backend

Optimizes application performance with caching strategies, query optimization, and Pulse monitoring setup.

Creates: config/cache.php, Pulse dashboards

laravel-deploy

DevOps

Configures deployment for Forge, Vapor, Docker, and Bref with zero-downtime strategies and health checks.

Creates: Dockerfile, serverless.yml

laravel-cicd

DevOps

Sets up CI/CD pipelines for GitHub Actions and GitLab CI with testing, linting, and deployment stages.

Creates: .github/workflows/

laravel-git

DevOps

Handles git operations including smart commits, pull request creation, and release management with changelogs.

Uses: git, gh CLI

laravel-testing

Quality

Generates Pest PHP tests - unit, feature, API, and browser tests with proper assertions and mocking.

Creates: tests/Unit/, tests/Feature/

laravel-security

Quality

Runs OWASP security audits, implements security headers, and fixes vulnerabilities with false positive filtering.

Checks: SQL injection, XSS, CSRF, mass assignment

laravel-review

Quality

Orchestrates code reviews with parallel specialized reviewers for security, quality, and Laravel best practices.

Delegates to: laravel-validator

laravel-refactor

Quality

Analyzes and refactors code for SOLID/DRY compliance, reducing complexity and improving maintainability.

Uses: Pint, PHPStan, custom rules
