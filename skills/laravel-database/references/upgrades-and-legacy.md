# Laravel/PHP Upgrades & Legacy Import

The original `laravel-database` skill did not contain upgrade or legacy-import material — those topics are owned by dedicated sibling skills. Use them directly:

- **Laravel version upgrades** (dependency bumps, breaking changes, deprecation fixes): `/laravel-agent:upgrade-laravel`
- **Legacy data import / migration from another system**: `/laravel-agent:migrate-from-legacy`

If you encounter upgrade or legacy-import work while working on database code, invoke the appropriate skill above rather than handling it here.
